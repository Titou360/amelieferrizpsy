# og-image.jpg — Image de partage social

Créer un fichier `og-image.jpg` dans ce dossier `/public/`.

## Spécifications requises

| Propriété | Valeur |
|-----------|--------|
| Nom | `og-image.jpg` |
| Format | JPG (pas PNG — trop lourd pour le partage) |
| Dimensions | **1200 × 630 px** (ratio 1.91:1) |
| Poids max | 300 Ko |
| Zone de sécurité | Contenu important dans les 900×450px centraux |

## Contenu recommandé
- Fond aux couleurs du site (navy `#1B2A4A` ou cream `#F0EDE8`)
- Logo `logo-amelie-ferriz-psychanalyste.png`
- Texte : "Amélie Ferriz — Psychanalyste à Narbonne"
- Sous-texte : "Psychanalyse · Psychothérapie · Thérapie de couple"
- Photo d'Amélie en second plan (optionnel)

## Outils gratuits
- Canva → template "Image de lien" 1200×630
- Figma → artboard personnalisé
